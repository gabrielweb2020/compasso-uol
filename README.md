# compasso-uol
API em Node JS e MongoDB desenvolvida como desafio da Compasso UOL.

#####
Express e Mongoose

## Instalação

#### Download do código
`https://github.com/gabrielweb2020/compasso-uol`  

#### Instalar Pacotes
`npm install`

#### Iniciar Base Mongo Local
`mongod`

#### Iniciar o Projeto
`yarn dev`

#### Documentação Swagger
http://localhost:3333/api-docs/
